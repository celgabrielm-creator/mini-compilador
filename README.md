# Mini Compiler

Compilador simples desenvolvido em Python.

## Funcionalidades

- Análise Léxica
- Análise Sintática
- Análise Semântica
- Tratamento de Erros
- Geração de Código Intermediário

## Executar

```bash
python main.py
```

## Exemplo

Entrada:

```txt
x = 10;
y = x + 5;
z = y + 20;
```

Saída:

```txt
x = 10
y = x + 5
z = y + 20
```
